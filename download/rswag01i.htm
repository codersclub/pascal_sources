RSWAG-2001 Install Instructions 


В этом выпуске содержится дополнение за 2001 год
к коллекции паскальных исходников Russian SWAG.
Внимание: здесь ТОЛЬКО НОВЫЕ ПОСТУПЛЕНИЯ за 2001 год!

Если ранее у Вас RSWAG не был установлен,
то данный пакет создаст коллекцию, содержащую
только новые поступления.

Для обновления уже установленной ранее версии Russian SWAG
выполните следующие действия:

1. Распакуйте архив RSWAG010.ZIP в директории, предназначенном
   для Russian SWAG. Мы рекомендуем, чтобы имя у него было
   \RSWAG.

                 pkunzip -d rswag010.zip

   В этом архиве содержится программа для обновления,
   база с описанием новых исходников за 2001 год,
   обновленный READER.EXE v4.3, и исправленный конвертор
   в HTML формат.

2. Загляните внутрь файла READER.INI, расположенного в
   каталоге RSWAG. Найдите значение переменной ListDir.
   (по умолчанию там написано ListDir = D:\HTDOCS )
   Измените по своему усмотрению это значение.
   В этом каталоге будут размещаться все ZIPованные
   исходники и там же при необходимости будут формироваться
   списки исходников в текстовом или HTML формате.

   Перепишите все остальные архивы (RSWAG011.ZIP...
   RSWAG01A.ZIP) в каталог, указанный в переменной ListDir,
   и разZIPуйте их:

     pkunzip -d RSWAG01*.ZIP

   Внимание: Обязательно распаковывайте с подкаталогами!!! 

   Эти 10 архивов содержат собственно исходники,
   разбитые по тематическим разделам.
  
3. Перейдите в директорий \RSWAG, содержащий RSWAG01.EXE
   и запустите его.

   Данная программа произведет дополнение имеющейся базы
   данных описаниями новых исходников за 2001 год
   (или создаст новую базу).

4. Для просмотра коллекции RSWAG запустите READER.EXE.

   Инструкцию по работе с программой READER.EXE Вы найдете
   в файле READER.TXT.

5. Обратите внимание, что в данном (RSWAG-2001) выпуске
   собственно исходники вынесены из базы данных в отдельные
   тематические подкаталоги для упрощения работы с ними!
