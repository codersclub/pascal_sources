1k 
 
                       DRVCOUNT - Freeware (BZSoft)

    Программа показывает количество доступных дисковых устройств.
    Предназначена для запуска в BAT файлах при использовании драйверов
    типа ADM.

    REM Пример запуска:
    DRVCOUNT > nul
    IF ERRORLEVEL==7 GOTO User0
    IF ERRORLEVEL==5 GOTO User1
    PATH = C:\;C:\DOS
    GOTO CONTINUE
    :User0
    PATH = C:\;C:\DOS;C:\PK;F:\TP;E:\ME
    GOTO CONTINUE
    :User1
    PATH = C:\;C:\DOS;C:\PK;E:\ME
    :CONTINUE
