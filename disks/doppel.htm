 Doppel 1.0 - Floppy Disk Duplicator for TP6  Unknown 27.07.1991

This program create a floppy disk image by copying every sectors to a hard disk. Then this diskimage have being copying to another diskette.



6k 
 
program doppel {Version 1.0};
{Zweck : Diskcopy von HD-Disketten,
         Mehrfachkopien ohne wechseln;
         Disketten muessen formatiert sein
         und duerfen keine schlechten
         Sektoren enthalten;
Sprache: Turbo Pascal 6.0
System : PC AT 286,386,486 unter DOS 3.x}
