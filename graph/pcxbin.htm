 PcxBin - Load and Show PCX Files   Chris Lamb 30.09.1988

Программа для вывода изображений в формате PCX, предварительно сконвертированных в объектный файл с помощью утилиты BinObj.
It will display a PCX file in many formats that is converted with BINOBJ and linked in at compile time.



10k 
 
A unit to load graphical pictures

                  

This is a conversion of the PCXSHOW program by myself and Dan
Davis. It will display a PCX file in many formats.  I have
changed the original to accept a pointer to a procedure.  This
procedure is a picture converted with BINOBJ and linked in at
compile time.  At this time you must also pass the size of the
file to the procedure.  I am trying to figure out how to find the
size of the linked procedure from within the program.  This would
enable you to link with any picture file and not worry what the
size is.

  Formats supported :
  BSAVE Pic Format                     320 x 200 x 004
  PC/Microsoft Paintbrush .PCX files : 320 x 200 x 004
  PC/Microsoft Paintbrush .PCX files : 640 x 350 x 016
  PC/Microsoft Paintbrush .PCX files : 320 x 200 x 256

I would appreciate you sending me any changes you make to the
code as I am always eager to learn new approches to problems.
