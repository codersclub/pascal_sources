      Данный пакет предназначен для вывода графических изображений из файлов
   формата PCX.

   В состав пакета входят:
      PCXLIB.PAS  -  Исходный текст на Паскале.
      PCXTEST.PAS -  Пример использования PCXLIB.PAS

   Примечания:

 1. Пакет разработан для работы с Turbo-Pascal ver. 6.0, в нем широко исполь-
    зуются директивы ASM ( Для увеличения скорости работы значительная  часть
    модуля написана на ассемблере ). Пользователям Turbo-Pascal ранних версий
    придется заменить директивы ASM на соответствующие директивы INLINE.

 2. Данный  пакет  разрабатывался для работы в составе  др. программ, поэтому
    в нем не производится тестирование текущего видеоадаптера ( чтобы не дуб-
    лировать код основной программы ),  а также  не  ведется  контроль ошибок
    файлового ввода/вывода.  При необходимости эти проверки легко вставить в
    указанные в тексте модуля места.
