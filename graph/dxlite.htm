 DirectX Lite unit v.0.10       Vadim Bodrov 25.03.2001


DirectX is a Microsoft DirectX 7 interface unit for TMT Pascal Multi-target (Win32). This archive includes a source code for the following DirectX interfaces: Direct Draw 7, Direct Input 7, Direct Sound 7, plus DirectX Utils and standard Microsoft DirectX sample applications ported to TMT Pascal.
