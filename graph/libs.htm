 Libs - SVGA Graphics Library from CoolSoft   Ivan Polyacov 05.06.2001

Libs.zip - набор графических библиотек для Borland Pascal 7.0. VGA, SVGA, DPMI...



29k 
 
Здесь представлен набор модулей для работы с графикой на Borland Pascal.
Все модули предназначены для 256-цветных VGA/SVGA режимов.

CURS13H.PAS   - Управление курсором в режиме 13h
CURS256.PAS   - Управление курсором в 256-цветных режимах
CURS256M.PAS  -
DPMI.PAS      - primitive routines for DPMI management
GRAPH13H.PAS  - Модуль Graph 13h для работы с режимом 320x200,256 цв.
MYGRAPH.PAS   - Графическая библиотека для Borland Pascal
                (Real and Protected mode)
                Поддерживает все 256-цветные режимы с LFB
SVGA.PAS      -
SVGA32L.PAS   - Графическая VESA 2.0 библиотека для Borland Pascal
                (Protected mode)
                Поддерживает все 256-цветные режимы с LFB
SVGADRV.PAS   - Модуль импорта для графической библиотеки SVGA32L
