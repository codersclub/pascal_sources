 PCXSHOW - Display PCX graphical formats   Dan T. Davis 04.08.1988

This program will display four types of graphical formats:
320 x 200 x 4 --- BSAVE format; try DEMO1.PIC and DEMO2.PIC
320 x 200 x 4 --- PCX format; try BUGS.PCX
640 x 350 x 16 --- PCX format; try PEPSI.PCX
320 x 200 x 256 --- PCX format; try MICKEY.PCX



21k 
 
It is a bare bones display; snow will probably appear on CGA monitors;
the program is intended to show how to throw graphical images on the
screen. If anyone speeds up the display (especially 640 x 350 x 16),
please let me know.
