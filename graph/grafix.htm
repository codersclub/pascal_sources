 2D Graphics for 320x200x256 Mode   Alex Frounze 15.09.1998

2-х мерная графика в режиме 320x200 256-цветов. Модуль (очень много разных функций) + примеры.



93k 
 
                       -= In Russian =-

                          Модули

GRAFIXX.PAS             Поддерживает режим 256-цветный режим
                        с разрешением 320x200
                        Ошизенное разнообразие процедур и
                        функций!
PACKER.PAS              Используется модулем GRAFIX.PAS
ROLLER.PAS              Для сдвига экранного изображения (Rolling/Scrolling)
GFXFONT.PAS             Содержит определение шрифта
GLIB.PAS                Модуль графического библиотекаря
KEYPRES2.PAS            Модуль для профессиональной работы с клавиатурой
FCMOUSE.PAS             Модуль для работы с мышью


  Программы, демонстрирующие возможности перечисленных модулей:

D3D2.PAS

GR_TEST.PAS

POLY.PAS
SCROLL'N.PAS
WAVES.PAS
VOB_VIEW.PAS            Визуализатор VOB-файлов
EGO.PAS                 Редактор VOB-файлов
GLIBMAN.PAS             Библиотекарь VOB-файлов (объединяет VOB-файлы в один
                        VLB-файл)
JERRY.PAS               Симпатичный мышонок, который умеет бегать
TANK.PAS                Ездящий и стреляющий танк

                        Другое:

GRIM.PCX                Картинка для программы CLIPPED.PAS
