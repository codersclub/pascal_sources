 MIP 1.00 - Modem Setup Program   Mark Billig 19.01.1991

Программа для инициализации последовательного порта. Работает только через Fossil драйвер.
MIP is a program for setting up your modem. A FOSSIL driver must be loaded first.



7k 
 
     MIP is a program that is for setting up your modem.
MIP *REQUIRES* that a FOSSIL be loaded first.

     This program is guaranteed to do nothing but take up your precious
hard disk space.

     MIP gets the information to send to the modem from a file called
MODEM.INT.
There is an example of this file in the archive.
