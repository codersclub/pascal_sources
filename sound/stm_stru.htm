     This file contain description of packed Scream Tracker module file
format. It is non-official format, but if you will use him in your ela-
borations he can become it. This description is accompanyed by a source
of input/output .STM & .CDM modules library for TurboPascal 6.0++. :)
(.CDM = [C]ompressed [D]igital [M]usic)

     This document contain the description for structure of .CDM module,
also describe an very simple algorythm of samples/patterns compression.
If you are not familiar with Lempel-Ziv/Huffman algorythm, you can try
to understand it, looking at program. In another case, you must search
for a clearly description of this algorythm.
