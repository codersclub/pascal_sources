 SMIX 1.25 - Sound Library with a Mixer for Real Mode   Ethan Brodsky 14.01.1995

Библиотека для Sound Blaster с поддержкой микширования. Может одновременно выводить до 8 различных звуков. Предназначена для Реального режима, но может быть переделана и для Защищенного режима.
(v1.25) SMIX - Sound Library (Turbo Pascal). Digital sound engine that plays multiple sounds simultaneously, mixing them in real time. Supports XMS, resource files, 16-bit output and auto-initialized DMA. Uses small fraction of CPU time. For Turbo Pascal real mode.
