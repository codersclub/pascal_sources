 Mini FM Organ Using Yamaha OPL-3 Chip  Roby Johanes 04.01.1998

Маленький одноголосый музыкальный инструмент для саундбластера. Клавиатура компьютера используется как музыкальная клавиатура. Программа автоматически распознает тип саундбластера и его параметры: базовый адрес, IRQ, канал DMA.
Mini FM Organ Using Yamaha OPL-3 Chip. Autodetect Sound Blaster using BLASTER Environment variable. You can use your PC keyboard instead of a piano.
