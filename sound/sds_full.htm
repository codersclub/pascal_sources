 SDS - Sound Deluxe System v5.02c   Maple Leaf 05.11.1997

Sound Deluxe System 5.02с, First public release. Компактная, стабильная и при этом мощная звуковая система с поддержкой различных звуковых карт и разных звуковых форматов.
SDS (Sound Deluxe System) is a "music interface kit" designed mainly for ASM+Pascal coders. It is able to load and play in background a module, using maximum 32 channels which are mixed software (by mixerHLL) or hardware (by GUS). It is therefore a good tool in developing and adding stunning music to your demos and games. It offers you a very easy interface in Pascal, which you are free (and therefore encouraged) to use in your programs. The SDS kernel runs entirely in one 64kb segment (code and own data). The loaders and the interface code run in another 64kb segment.

  - Powerful, stable and compact sound system
  - Internal patterns compression (5:1)
  - EMS support (up to 31 Mb)
  - UMB support (up to 64 Kb)
  - 32 channels, 254 instruments, 254 patterns
  - X-Rows patterns
  - Real-time amplification/reduction
  - Stereo where available
  - Dolby Surround where available
  - 16-level panning in stereo
  - PAL/NTSC mode option
  - API services
  - External Syncronization Block (ESB)
  - Support for: SB, SB 2.0, SB Pro, SB16, 1Mb GUS, PAS, WSS, Aria and Silence
  - Module loaders: MOD, STM, S3M, MTM, 669, FAR, ULT, SDM
  - Module savers: SDM
  - Already-built detection/configuration/init interface (Pascal)
  - A lot of example source code
  - Full documentation!
