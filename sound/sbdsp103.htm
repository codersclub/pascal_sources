 SBDSP v1.03 - Direct Sound Blaster Programming   Ethan Brodsky 23.09.94

Модуль и программа для проигрывания VOC файлов на саунд-бластере. Это полный функциональный аналог модуля SBVox, но вместо использования драйвера CT-VOICE.DRV используется непосредственное прямое программирование бластера.
SBDSP performs the same functions as the SBVox unit but WITHOUT using the CT-VOICE driver. It has only been tested on a SB16 and PAS16, but it should work on all Sound Blaster compatible sound cards. By using DMA transfers, it plays sound without using the CPU, saving processor cycles for your program.
