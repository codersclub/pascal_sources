Tandy DAC Sound Package
Version 1.6
July 29, 1994
Jeffrey L. Hayes

This package contains programs and programming information for the
proprietary digital-to-analog converter circuitry included with several
models of the Tandy 1000 and 2500.  For usage information on the programs,
see Tspak.doc.  For programming information, see Tandac.doc.

The following files are included:

README         This file.
HISTORY        Version history.
BUGFIX         Bugs to be fixed (some of them).
TSPAK    DOC   Usage information on programs in this package.
TANDAC   DOC   Programming information file.
PIANO    DOC   Documentation for Piano (for older models without the DAC).
DIVIDER  DOC   Experimental data for DAC divider values.
SND      FOR   Tandy DeskMate .snd file format description.
RECWAV   EXE   RIFF WAVE recorder program.
RECWAV   PAS   Main source code file for Recwav.
DO_REC   ASM   External assembler subroutine for Recwav.
DAC_INIT ASM   External assembler subroutine for Recwav.
TIME_DIV ASM   External assembler subroutine for Recwav.
RECWAV   MAK   Text instructions for compiling Recwav.
PLAYWAV  EXE   RIFF WAVE player program.
PLAYWAV  ASM   Source code for Playwav.
PLAYWAV  MAK   Text instructions for assembling Playwav.
RAW2WAV  COM   Utility to convert 8-bit raw PCM files to RIFF WAVE format.
RAW2WAV  ASM   Source code for Raw2wav.
RAW2WAV  MAK   Text instructions for assembling Raw2wav.
PPWAV    EXE   RIFF WAVE preprocessor, reduces .wav file size.
PPWAV    ASM   Source code for Ppwav.
PPWAV    MAK   Text instructions for assembling Ppwav.
SND2SAM  EXE   Utility to convert Tandy DeskMate .snd files to Amiga .mod
               samples.
SND2SAM  PAS   Source code for Snd2sam.
SND2SAM  MAK   Text instructions for compiling Snd2sam.
SETDAC   COM   Utility to place the DAC
