  Welcome to BWSB. This is a programming library for adding full
digital music and sound effects to your own programs! It is fast,
compact, and easy to learn.

  With this package you could create a first class game like OMF
2097. You'd be surprised how much music and sound effects add to
a game. If you've seen both OMF 2097 and the original OMF, you
know the difference. You could also create demos like Unreal or
Panic (BWSB can even play the music found in those demos!). There
are also many other uses for BWSB that you can come up with
yourself, so get thinking!

  To see some of what BWSB can do in action, run the GO.BAT batch
file. This just plays a little music on your sound card. If you're
ready to see some more, run some of the other EXE files included
here. If you decompressed BWSB with PKUNZIP and the -D (create
directories) parameter, you can find all the example programs in
the \DEMO directory.
