 dbBrowse 2.0 - Simple dBase Browser   Mike Shunfenthal 03.07.1988

Простая программа для просмотра баз данных dBase. Позволяет выбрать файл из списка, показывает структуру базы, позволяет "пролистывать" вперед-назад записи.
