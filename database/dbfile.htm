 dbFile. ООП-библиотека для работы с DBF.  Илья Чубаров 28.03.1994

Объектная библиотечка для работы с DBF файлами на TurboPascal 7.0-6.0. Поддерживаются операции: Добавление, Удаление и Вставка записи, Доступ к полю записи, Фильтрация. + Перемещения по базе. Отсутствуют: Сортировка, Pack и Индексирование.

Господа!  У меня имеется объектная библиотечка, собственной
разработки, для работы с DBF файлами на TurboPascal 7.0 6.0 .
Она позволяет создавать свои DBF и использовать уже созданные.
Поддерживаются операции:
Добавление, Удаления и Вставка записи, Доступ к полю записи,
Фильтрация. + Перемещения  по базе.
В скором времени (как только появится время) напишу Сортировку,
Pack и поддержку индексирования по B-Дедевьям.
