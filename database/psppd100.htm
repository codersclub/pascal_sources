 PSPPD 1.0 - Database functions for Turbo Pascal 6.0   Michael Gallias 08.01.1993

Метод создания базы ресурсов, необходимых программе, в виде "файла файлов". Такой метод позволяет не плодить множество маленьких файлов (спрайтов, иконок, справок...), а разместить их все в одном-единственном файле, а затем получать доступ к каждому из них внутри программы.
Advanced database support. Allows you to store files in files. This saves disk space and is ideal for storing many little files (like icons). You can request a file from the database as if it were a normal file and thus makes programs more professional without major code modification.
