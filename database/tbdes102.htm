 TbDes 1.02 - Print Paradox 4 Tables Structure   Salvatore Besso 27.08.1994

Программа для контроля целостности структуры баз данных формата Paradox 4. Печатает структуру базы и возвращает ErrorLevel. Для работы требуется Paradox Engine 4.x.
Program TablesDescription Prints Paradox 4 tables structure and returns an Errorlevel if any error occured. The Paradox Engine 4.x required.
