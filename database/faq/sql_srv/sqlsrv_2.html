Как сделать выборку данных из нескольких таблиц
SQL позволяет в одном запросе обращаться к нескольким таблицам. Полное имя столбца в таблице фактически состоит из имени таблицы, за которым идет точка и имя столбца. 
SELECT Table1.field1, Table2.*
    FROM Table1, Table2
