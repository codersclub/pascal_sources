Где взять информацию о таблицах, колонках, индексах, etc.
В SQL Server вся информация подобного рода хранится в системных таблицах - SYSOBJECTS, SYSCOLUMNS, SYSINDEXES и других. Например, для получения информации о пользовательских таблицах выполните запрос 
SELECT * FROM sysobjects WHERE type = 'U'
