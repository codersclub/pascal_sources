Как создать/изменить таблицу
Для создания таблицы нужно выполнить запрос:
CREATE TABLE Table1 (
    id int(10) not null auto_increment,
    text_field text,
    int_field int(10) not null DEFAULT 0,
    PRIMARY KEY (id))
Этот запрос создаст таблицу Table1 с ключевым полем-счетчиком id и двумя полями text_field и int_field,, причем последнее имеет значение по умолчанию 0.

А теперь, если нам нужно изменить таблицу (например, добавить столбец), выполняем такой запрос:
ALTER TABLE Table1 ADD new_row int(10)
Этот запрос добавил колонку new_row типа int. А теперь удалим этот столбец:
ALTER TABLE Table1 DROP COLUMN new_row
А если нужно удалить всю таблицу, достаточно выполнить такой запрос:
DROP TABLE Table1
