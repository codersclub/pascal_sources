Как использовать БД MySQL в своей программе
Как я уже говорил, для MySQL есть интерфейсы для разных языков - C, Perl, PHP. В В любом случае, нужно найти заголовочные файлы для нужного языка и включить их в проект. Пример для VС++: нужно указать линкеру путь к библиотеке libMySql.lib (Tools->Options->Directories) и заголовочным файлам (все эти файлы идут в дистрибутиве с MySQL). Теперь скопируйте libMySql.dll в системный каталог Windows. Все, теперь можно использовать MySQL API в вашей программе. Кстати, описание API и примеры тоже есть в дистрибутиве.
