Как посмотреть структуру БД/таблицы
Да... такой вопрос вызвал бы у пользователей SQL Server'а только смех... но нам не до смеха :( Так вот, в MySQL для этого есть команда SHOW. Вот что можно получить с ее помощью:
SHOW DATABASES
SHOW KEYS FROM table_name
SHOW TABLES [FROM database]
SHOW [COLUMNS|FIELDS] FROM table [FROM database]
Думаю, комментарии тут излишни :)
