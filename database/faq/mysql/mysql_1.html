Как подключиться к MySQL из Access (и не только)
MySQL - очень быстрый бесплатный сервер БД, да еще работающий на разных ОС (Linux, Windows). Конечно, он не лишен недостатков - в нем отсутствуют триггеры, хранимые процедуры, транзакции, подзапросы, визуальная среда разработки и управления, но тем не менее MySQL идеально подходит для средних по сложности проектов с большой нагрузкой на сервер (например, БД для веб-сайтов).

Для подключения к MySQL-серверу есть отдельные интерфейсы для основных языков программирования, используемых для разработки CGI (что подчеркивает веб-ориентацию сервера) - C, Perl, PHP. Но согласитесь, что создавать БД с десятком полей с консоли не очень удобно :), а свой интерфейс писать - долго. Вот поэтому и был создан драйвер MyODBC, который позволяет другим приложениям работать с MySQL сервером. Вот как это сделать для MS Access:
1) Cкачиваем MyODBC и установливаем его (найти можно на  http://www.mysql.com). Настраиваем DSN для нашего MySQL сервера.
2) В Access открываем базу данных и давим на "Создать", затем "Связь с таблицами". Указываем в качестве типа файлов указываем ODBC и выбираем созданный в предыдущем шаге DSN.
3) Для обновления таблиц можно воспользоваться утилитой "Сервис"->"Служебные программы"->"Диспетчер связанных таблиц".
