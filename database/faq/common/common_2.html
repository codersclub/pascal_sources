Какой сервер БД выбрать?
Спорных тем, подобных этой, существует немало; и навязывать, кому с каким сервером работать я вовсе не собираюсь. Поэтому я приведу только факты, а уже опираясь на них вы сами сделаете выбор. Итак:

Сервер Базовая ОС Визуальная среда разработки Триггеры Хранимые процедуры Соответствие
ANSI SQL92		 Скорость (max. 5)*
InterBase Windows есть есть есть полностью 4.5
MySQL Linux, Windows нет нет нет частично 4.9
Oracle Linux, Windows есть есть есть полностью 5
SQL Server 2000 Windows есть есть есть полностью 4

* Оценка скорости приведена по тестам eWeek и другим источникам. Кроме того, средняя скорость сервера определяется по множеству параметров - скорости блокировки, обработки транзакций и др., т.е. по отдельным параметрам картина может быть иная.
