 H_Disk 1.0. Закрытие доступа к диску  Сергей Гребенюк 23.01.1995

Программа для ограничения доступа к жесткому диску в Ваше отсутствие



5k 
 
Bessel club product                                    S SYSTEM
                           H _ D I S K

                             VER 1.0


    Предлагаем Вам  программу  для ограничения доступа к жесткому
диску в Ваше отсутствие.
    Запуск программы осуществляется при помощи командной строки:
    H_Disk on - подключение жесткого диска для использования.
    H_Disk off - отключение жесткого диска для использования.
    При запуске программы с ключом off у пользователя  запрашива-
ется  пароль,  с  которым  он желает ограничить доступ к твердому
диску.  Для исключения ошибок программа требует  подтвердить  па-
роль.  После  этого  происходит изменение служебной информации на
винчестере.  При новой загрузке операционной системы BIOS распоз-
нает  винчестер  как аварийное устройство и не подключает его для
дальнейшего использования.
    Для использования  винчестера  запустите  программу  H_DISK с
ключом on с дискеты. В данном режиме у пользователя запрашивается
пароль,  сравнивается с паролем, хранящимся на винчестере. В слу-
чае совпадения твердый диск подключается для использования, прог-
рамма выдает сообщение:  Операция завершилась успешно.  Далее Вам
требуется перегрузить систему.

    ВНИМАНИЕ! Обязательно   храните  копию  программы  H_DISK  на
дискете.

    Состав программного обеспечения:
    H_DISK   PAS    21134   9-23-91   9:12p
    H_DISK   TXT     2048   9-23-91   9:12p
