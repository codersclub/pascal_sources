   Программа предназначена для блокирования трех операций с файлами.
Удаление, запуск и открытие файла не выполняются, если в конфигурационном
файле указанная операция запрещена.
   Для установки значенй в CFG файле используется программа LSETUP.

Программа имеет инуитивный интерфейс, но на всякий случай кое-какие замечания:
1. Чтобы добавить элемент в список используйте Alt+A ил кнопка <Add> - 
   добавляет следующим образом - если есть отметки файлов, то добавляет эти 
   файлы, если нету, то добавляет то что в строке ввода <Имя файла>.
   Двойной щелчок мыши в списке файлов - добавляет этот файл. 
   Чтобы поставить отметку на имя файла нажмите правую кнопку мыши или
   клавишу Insert.
2. При вставке элемента в список используются текущие
