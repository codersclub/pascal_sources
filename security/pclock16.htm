 PCLOCK.SYS включается в CONFIG.SYS : DEVICE=PCLOCK.SYS
Он защищает себя, CONFIG.SYS и три директории (или файлы),
имена которых устанавливаются инсталлятором INST_PCL.EXE.
Все изменения действительны только после перезагрузки машины.
При запуске инсталлятор запрашивает пароль SuperUser.
 При инсталляции можно защитить жесткий диск - при загрузке
с дискеты он будет недоступен на логическом уровне.
Для этого на вопрос "Install PC Lock partition ?" ответьте "Yes".

 Два уровня защиты :
  1) Открываются только дисководы, закрытые директории
     остаются закрытыми - пароль 'drive password'.
  2) Пароль SuperUser - защита снимается полностью.

 Для включения защиты вводится неправильный пароль.

 По умолчанию :
  SuperUser password : 'rabbit',
  Drive password : 'unlock',
  First directory : 'GAME',
  Second directory : '',
  Third directory : ''.

 Вызов - нажатием hot-key 'Alt_Tab'.
