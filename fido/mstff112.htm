 MsgStuff v1.12 - Jam, Squish, Msg, Pkt2.0+ Converter   Dmitry Suhodoev 12.06.2001

MsgStuff - небольшой набор утилит для продвинутого фидошника :-) содержит в себе конвертер фидошных баз, которая может конвертить из любого из следующих форматов в любой: squish, jam, msg (fido/opus), pkt 2+. Дополнительно прилагается сортировщик сообщений по дате написания для каждого из этих форматов баз. Компилируется под BP7 и VP2+ для win32, os/2, dos16, dpmi32, dpmi16.
