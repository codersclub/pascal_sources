маленькая yтилка для бyковки "H" (года два назад сделал) :)...

Программка заменяет во всем файле символ "H" русскую (код 141) на
символ "H" латинскую (код 72).
Измененный файл с буквой "H" (латинская) сохраняется с расширением
"_H_", то есть файл srn090.prt будет файлом srn090._h_!
Программка ИЗМЕHЯЕТ ЛЮБЫЕ, файлы будь-то .EXE .COM или .TXT .PRT !!!
