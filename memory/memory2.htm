 Allocate Memory Block lager than 64K   Alex Frounze 08.11.1998

Процедуры для резервирования и освобождения памяти блоками большего размера чем 65520 байт (предел компилятора). Модуль + пример.



1k 
 
                          -= In Russian =-

        MEMORY.PAS              Модуль для взятия памяти не через менеджер
                                кучи, а через функции DOS. Позволяет брать
                                блоки, не ограниченные размером 65520 байт.

        MEM_TEST.PAS            Примитивнейший пример использования модуля.

                                                      Автор: Фрунзе Алексей


                          -= In English =-

        MEMORY.PAS              This unit allow you to allocate memory
                                blocks greater than 65520 bytes. Used
                                DOS functions not a heap manager's.

        MEM_TEST.PAS            Simple example of using Memory unit.
