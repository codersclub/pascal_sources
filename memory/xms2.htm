 XMS Support Unit   Alex Frounze 23.07.1998

Использование XMS для хранения данных посредством драйвера HIMEM.SYS. Модуль + пример.



2k 
 
                          -= In Russian =-

          Если Вы задыхаетесь в 640 килобайтах памяти реального
режима, а Вам хочется хоть как-то использовать имеющуюся память
компьютера, то вспомните о старом добром драйвере HIMEM.SYS!

XMS.PAS         Модуль для работы с XMS-памятью

XMSUSING.PAS    Програмулька, демонстрирующая пересылку блоков памяти
                (копирует содержимое экрана в XMS-память и очищает дисплей;
                ждет нажатия кнопки, после которого восстанавливает
                изображение экрана из XMS-памяти)

                                                     Автор: Алексей Фрунзе


                          -= In English =-

          If you have an asphixia in the 640 KB of DOS memory
        remember that you have an old but good HIMEM.SYS driver!

XMS.PAS         Main Unit (acces to the XMS-memory with HIMEM.SYS)

XMSUSING.PAS    Example of memory blocks moving -
                Copies screen to the XMS and back from the XMS
