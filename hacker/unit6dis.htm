To run the unit disassembler, type UNIT6DIS at the C:\> prompt in
the directory of the Turbo Pascal 6.0 unit you want to disassemble.
When asked for the name of the unit,  type it in, but -=WITHOUT=-
the .TPU extension.  The unit must be in the current directory or
in some directory on the current path, either as an individual unit,
or one in "TURBO.TPL".  You will then be asked whether or not to
disassemble the object code.  -=WARNING=-!  SYSTEM.TPU will produce
over 700-800K of output in the list file.
