  This unit provides a quick way to make a bounce bar menu.  You
  call it as follows:

  Init object, (TopLeftCol, TopLeftRow, Length : Integer;
                Title : String;
                BackgroundColour, HilightColour, Frame style : BYTE;
                AllowLEFT/RIGHTtoEXIT)  :  Boolean;

  Frame Styles  0-4

  AllowLEFT/RIGHTtoEXIT :  if left arrow key exits, return is -1.
                           if right arrow key exits, return is -2.

  If ESC is pressed return is 0.

See MENUDEMO.PAS for a good example on how to use it.
