  VBU v.1.0 - Vitus Batch Utilites.
  Набор программ для создания менюшек и окошек в DOS-овских командных файлах.
  Поставляется в исходных текстах. Требует Turbo Professional.

  Состав файлов:

  makefile.    - файл для пакетной компиляции VBU 1.0
  getfile.pas  - утилита поиска файла по маске, возвращает имя и ErrorLevel
  gfhelp.txt   - Help для getfile
  menu.pas     - меню, возвращает ErrorLevel
  menuhelp.txt - Help для menu.pas
  visit.pas    - утилита смены текущего директория с возвратом назад
  vhelp.txt    - Help для visit.pas
  window.pas   - Рисование окошек
  winhelp.txt  - Help для window.pas
  xset.pas     - утилита для установки переменной окружения DOS
