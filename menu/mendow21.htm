The MENDOW21.INC Routines Allow A User To Easily Create WINDOWS &
MENUS.  Both The Windows & the Menus have a pleasing double line
border around them.  MENDOW21 also allows Windowed Data Entry
Screen Generation, but that feature is not fully implemented as
of this writing. Heres How You Use MENDOW21.INC:

The new version of MENDOWnn (MENDOW21) has been expanded to allow
for moving menus.  This feature forced me to change the old
versions Make_Menu routine passing parameters so if you use
MENDOW10 you should read the section on MENU Creation in the
MENDOWS.DOC file.  I have also added a programming example that
should help clear up any confusion.
