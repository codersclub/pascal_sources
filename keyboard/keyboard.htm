Клавиатура

 Keyboard Handler Unit       Ruslan Fazliev 15.01.1997


Этот модуль полностью перехватывает на себя управление клавиатурой.
Позволяет отслеживать одновременное нажатие сразу нескольких клавиш.
Например, в демо примере выход осуществляется по одновременному начатию сразу шести клавиш -QWERTY
