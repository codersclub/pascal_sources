 DateRout - Simple Date Routines  John Wood 01.04.1988

Набор функций для работы с датами:
Сравнение и сложение дат, номер и имя дня недели, определение високосного года.



4k 
 
                                                        3/28/1988

   This is my second try at DateRout.  The first version contained
 an incorrect assumption regarding the quadyear count.  Essentially,
 DaysFrBase and ConvertDays assumed continuous 1461 day segments
 within a four century span.  This version corrects that by:
   1. changing the BaseYr to 1600.
   2. adding a century counter.
   3. adapting the quadyear count to work correctly within century's.

   If there are any further problems with this code (or suggestions
 for improvement), please write or leave a message to me on
 the PC Library BBS - Minneapolis (612) 435-1785.

 you to do the same with your programming gems.  Use it however you
