1. OVERVIEW
------------

Kuba_MSE is a  Borland Pascal unit  package  containing  both
real and protected  mode libraries  (respectively  KUBA_MSE.TPU
and  KUBA_MSE.TPP).  It contains a  compact set of all  necessary
mouse  handling functions  (based on interrupt 33h).

2. FILES IN THIS PACKAGE
------------------------

  KUBA_MSE.DOC    -       this file
  KUBA_MSE.PAS    -       source code
  KUBA_MSE.TPU    -       BP7 real mode unit
  KUBA_MSE.TPP    -       BP7 protected mode unit
  MDEMO.PAS       -       BP7 Kuba_MSE demo program
