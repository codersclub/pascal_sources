BASS
Звуковая подсистема для Windows 9x/NT/2K.
Предоставляет разработчику набор мощных, но простых в использовании средств. Обеспечивает:
- поддержку звуковых потоков (MP3, WAV, сгенерированные случайным образом)
- работу с MOD файлами (XM, IT, S3M, MOD, MTM)
- работу с MO3 файлами (MOD файлы, сжатые по MP3)
- проигрывание аудио CD (playback functions).
- 3D позиционирование сэмплов, потоков и MOD музыки
- поддержку A3D и EAX.
http://www.un4seen.com/music/music_bass.html
