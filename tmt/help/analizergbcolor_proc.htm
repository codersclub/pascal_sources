AnalizeRGBColor procedure

Targets: MS-DOS, Win32
Graph Unit

Returns RGB fields of given color.

Declaration: 
procedure AnalizeRGB(Color: DWORD; var R,G,B: Byte);
Remarks:
This function works in HiColor and TrueColor SVGA modes only. RGB field values, returned by AnalizeRGB depended on the current video mode (32K, 64K or 16M colors).
