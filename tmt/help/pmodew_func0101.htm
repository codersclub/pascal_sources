Function 0101 - Free DOS Memory Block

PMODE/W API

This function Frees a low memory block previously allocated by function 0100h. 
In:
   AX     = 0101h
   DX     = protected mode selector for memory block
Out:
   if successful:
     carry flag clear
   if failed:
     carry flag set
     AX     = DOS error code
