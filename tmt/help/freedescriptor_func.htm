FreeDescriptor function

Targets: MS-DOS only


DPMI Unit

This function is used to free descriptors that were allocated through the AllocateDescriptors function.

Declaration: 
function FreeDescriptor(Selector: Word): Boolean;
Remarks:
Returns True if successful.
