GraphDefaults procedure

Targets: MS-DOS, Win32


Graph Unit

Homes the current pointer (CP) and resets the graphics system to specified default values.

Declaration: 
procedure GraphDefaults;
Remarks:
Homes the current pointer (CP) and resets the graphics system to the default values for viewport, palette, draw and background colors, line style and line pattern, fill color, active font, text style, text justification, and user Char size.
