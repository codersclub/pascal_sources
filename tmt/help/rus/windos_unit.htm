Standard Units

The WinDos Unit

Targets: MS-DOS, OS/2, Win32


The WinDos unit allows easy access to most of the functions provided by the MS-DOS operating system from 32-bit protected mode application. Also WinDos unit emulates MS-DOS functions under OS/2 and Win32 using standard API, provided by OS/2 and Win32 operating systems. Operations such as find file, disk size or status, time and date, get environment strings, and more are provided by the WinDos unit. It provides a PChar interface to the file handling functions.
