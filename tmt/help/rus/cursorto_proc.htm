CursorTo procedure

Targets: Win32 GUI


WinCRT Unit

Moves the cursor to the given coordinates within the WinCRT window.

Declaration: 
procedure CursorTo(X, Y: Longint);
Remarks:
CursorTo(X,Y) is equivalent to GotoXY(X + 1, Y + 1).

See also: GotoXY
