Frac function

Targets: MS-DOS, OS/2, Win32

System Unit

Returns the fractional part of the argument.

Declaration: 
function Frac(X: Extended): Extended;
Remarks:
Frac simply subtracts the integer part of X from X, i.e. Frac(X) = X - Int(X).
