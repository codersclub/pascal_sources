DateTime type

Targets: MS-DOS only


Dos Unit

The UnpackTime and PackTime procedures use variables of type DateTime to examine and construct 4-byte, packed date-and-time values for the GetFTime, SetFTime, FindFirst, and FindNext procedures:

Declaration: 
type
  DateTime = record 
    Year, Month, Day, Hour,
    Min, Sec: Word;
  end;
