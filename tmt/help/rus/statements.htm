TMT Pascal Language Description
Pascal Language Structure

Statements

A statement indicates the action a program performs. Statements are separated by semicolons (;). Statements may be preceded by a label which consists either of an identifier or an unsigned integer constant.
  • Assignments
  • Compound Statements
  • Case Statement
  • For Statement
  • Goto Statement
  • If Statement
  • InLine Statement
  • Mem, MemW, MemL, and MemD Statements
  • Port, PortW, and PortD Statements
  • Repeat Statement
  • While Statement
  • With Statement
