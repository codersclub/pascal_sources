Type Declarations

Type declarations are preceded by the Type reserved word. 
Type
  identifier = Identifiertype;
Identifier is the actual name of the new type you define. Identifiertype is the type of identifier. Identifiertype can be on of the following:
  • array
  • File
  • function
  • interface
  • object
  • Pointer
  • procedure
  • record
  • set
  • String
  • Text
  • Another Type
  • Subrange
  • Enumeration
Some examples of type declarations are: 
type
  Float        = Extended;
  Int          = Integer;
  Filename     = array [0..8] of Char;
  Fnameptr     = ^Filename;
  Seasons      = (Winter, Spring, Summer, Autumn);
