Standard Units

The Use32 Unit

Targets: MS-DOS, OS/2, Win32


Contains redefinition of integer types for 32-bit computing as follows: 
type
  Word       = System.DWORD;
  PWord      = ^System.DWORD;
const
  MAXWORD    = High(System.DWORD);
  MINWORD    = Low(System.DWORD);
