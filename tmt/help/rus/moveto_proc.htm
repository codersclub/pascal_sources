MoveTo procedure

Targets: MS-DOS, Win32


Graph Unit

Moves the current pointer (CP) to (X,Y).

Declaration: 
procedure MoveTo(X, Y: Longint);
Remarks:
The CP is similar to a text mode cursor except that the CP is not visible. The following routines move the CP:
  • ClearDevice
  • ClearViewPort
  • GraphDefaults
  • SetGraphMode
  • SetSVGAMode
  • LineRel
  • LineTo
  • MoveRel
  • MoveTo
  • OutText
  • SetViewPort
