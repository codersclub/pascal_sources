ZenTimer Unit Procedures and Functions

ULZTimerCount function

Targets: MS-DOS, OS/2, Win32


ZenTimer Unit

Returns the current count for the Ultra Long Period Zen Timer.

Declaration: 
function ULZTimerCount: DWord;
Remarks:
Returned value is a current count that has elapsed between calls to ULZTimerOn and ULZTimerOff in resolution counts.
