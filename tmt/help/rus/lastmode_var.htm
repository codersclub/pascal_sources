Crt Unit
CRT Unit Variables

LastMode variable

Targets: MS-DOS, OS/2, Win32 console


CRT Unit

Each time TextMode is called, the current video mode is stored in LastMode.

Declaration: 
var LastMode: Word;
Remarks:
Also, LastMode is initialized at program startup to the then-active video mode.
