TMT Pascal Language Description

Pascal Language Structure

TMT Pascal programs are to be written either the TMT Pascal IDE editor or an editor of your choice. The source files created by your editor must be standard ASCII text. All characters within the range of 32 to 127 (decimal) are valid. Control characters (characters below 32 decimal) are treated as spaces.
