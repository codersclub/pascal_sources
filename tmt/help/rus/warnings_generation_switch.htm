Compiler Directives
Switch and Parameter Directives

$W: Warnings Generation Switch

Targets: MS-DOS, OS/2, Win32


Compiler directives

Switches on/off warnings generation.

Syntax: 
  {$W+} or {$W-}
Default: 
  {W+}
Remarks:
If {$W+} defined, TMT Pascal will show compilation warnings.
