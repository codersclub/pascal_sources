VbeInfoType

Targets: MS-DOS only


Graph Unit

A record that stores the VESA VBE information block; used by GetVbeInfo.

Declaration: 
 VbeInfoType         = record 
   VbeSignature       : DWORD;
   VbeVersion         : Word;
   OemStringPtr       : DWORD;
   Capabilities       : DWORD;
   VideoModePtr       : DWORD;
   TotalMemory        : Word;
   OEMSoftwareRev     : Word;
   OEMVendorNamePtr   : DWORD;
   OEMProductNamePtr  : DWORD;
   OEMProductRevPtr   : DWORD;
   Reserved           : array [0..221] of Byte;
   OEMData            : array [0..255] of Byte;
 end;
