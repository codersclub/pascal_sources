IDE (Integrated Development Environment)

Bookmarks

The Code Editor supports up to 10 bookmarks. Set your own bookmarks by right-clicking in the Code editor and choosing the Tggle Bookmarks. To jump to a bookmark, right-click and choose Goto Bookmarks.

The following bookmark operations shortcuts are available:
