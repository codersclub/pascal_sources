Standard Units
MMedia Unit
TMMedia Object Methods

TMMedia.Close

Targets: Win32 only


MMedia Unit

Closes the open multimedia device.

Declaration: 
  destructor Close;
Remarks:
The StopAtClose field determines whether currently playing media is stopped when the Close method is called. The Wait field determines whether control is returned to the application before the Close method has finished. The Notify field determines whether Close generates a MM_MCINOTIFY message.
