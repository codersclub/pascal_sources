Win32MajorVersion function

Targets: Win32 only


System Unit

This function returns the Win32 major version.

Declaration: 
 function Win32MajorVersion: Longint;
Remarks:
Returns the major version number of the operating system.
For example, for Windows NT version 3.51, the major version number is 3; and for Windows NT version 4.0, the major version number is 4.
