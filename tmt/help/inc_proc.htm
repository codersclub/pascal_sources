Inc procedure

Targets: MS-DOS, OS/2, Win32


System Unit

Increments a variable.

Declaration: 
procedure Inc(var  X [ ; N: Longint] );
Remarks:
If N is not passed to Inc, X is incremented by one. Otherwise, Inc(X) is identical to X := X + n or X +:= n.

Example: 
  Inc(Int,7);  // Int  := Int + 7
  Inc(Long);  // Long := Long + 1
