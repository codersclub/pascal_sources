Standard Units

The DOS Unit

Targets: MS-DOS, OS/2, Win32


The Dos unit allows easy access to most of the functions provided by the MS-DOS operating system from 32-bit protected mode application. Also Dos unit emulates MS-DOS functions under OS/2 and Win32 using standard API, provided by OS/2 and Win32 operating systems. Operations such as find file, disk size or status, time and date, get environment strings, and more are provided by the Dos unit. In total, over 50 functions and procedures are available. For more information about MS-DOS consult your DOS operating system reference manual.
