TMT Pascal - 32-битный компилятор языка Pascal
для DOS32, Win32, OS/2, Linux

С помощью компилятора TMT Pascal Вы можете легко построить свои 32-битные приложения. Он поможет Вам вдохнуть новую жизнь в Ваши старые MS-DOS приложения и покажет Вам самый простой путь к созданию программ для Microsoft Windows' 95/98/Me/NT/2000/XP и IBM OS/2. Возьмите свои старые исходники на Turbo Pascal и без каких-либо модификаций просто перекомпилируйте их с помощью TMT Pascal Multi-Target Edition для получения 32-битных консольных Windows или OS/2 приложений. Забудьте о "Runtime Error 200" - ошибке, связанной с модулем CRT в Turbo Pascal, и появляющейся на современных процессорах типа Pentium, Celeron, Pentium II и Pentium III. Наш модуль CRT работает стабильно на любых процессорах и может использоваться для всех поддерживаемых компилятором платформ (MS-DOS, OS/2 и Microsoft Windows' 95/98/Me/NT/2000/XP).
Вы можете испытать DOS-версию компилятора бесплатно!

В состав TMT Pascal входят мощные библиотеки для поддержки технологий Silicon Graphic OpenGL и Microsoft DirectX 8.0. Эти программные интерфейсы (API) сециально предназначены для разработки игр и других высокопроизводительных мультимедийных приложений. Они включают поддержку двумерной (2D) и трех-мерной (3D) графики, музыки, звуковых эффектов, устройств ввода и т.д.

Широко известная графическая библиотека
SciTech Multi-platform Graphics Library (MGL) полностью портирована в TMT Pascal.
Используя библиотеку SciTech MGL, вы сможете без значительных затрат времени создавать свои собственные интерактивные графические приложения, нисколько не уступающие таким монстрам, как WinQuake, Hexen II и многим другим, построенным с помощью именно этой библиотеки.
Подробнее о самой библиотеке вы сможете прочитать на www.scitechsoft.com/dp_mgl.html (engl).

TMT Development Corporation является партнером компании Microsoft по разработке и поддержке корпоративной платформы для Интернета нового поколения Microsoft .NET начиная с самого начала ее развития. Подробнее об участии TMT в проекте.NET (engl).

В состав компилятора TMT Pascal входит мощная графическая библиотека, использующая спецификацию VESA/VBE 2.0 (включая поддержку LFB) для MS-DOS, и технологию Microsoft DirectDraw для Windows приложений. Эта библиотека позволит Вам создавать очень быстрые графические 32-битные приложения как для для MS-DOS (в защищенном режимe), так и для Windows. Библиотека максимально приближена к модулю Graph из Borland Pascal. Хотя в это верится с трудом, но теперь Вы сможете перекомпилировать Ваши старые Pascal исходники, превратив их в современные 32-битные MS-DOS или Win32 GUI приложения! В большинстве случаев Вам даже не потребуется вносить никаких изменений в исходники, чтобы создать из них настоящие Win32 DirectX задачи. Графические приложения, написанные на Pascal, еще никогда не были такими быстрыми и мощными!.

Assembler, встроенный в TMT Pascal, прекрасно управляется с новейшей технологией Intel(tm) MMX (MultiMedia eXtensions), которая поддерживается современными процессорами Pentium, Pentium II и Pentium III. Эта технология предназначена для ускорения работы графических и мультимедийных приложений. Попробуйте наш компилятор для создания Ваших собственных приложений, оптимизированных под MMX. Подробнее о технологии MMX (engl).

Начиная с версии 3.20, встроенный в TMT Pascal ассемблер поддерживает технологию AMD(tm) 3DNow!, котора является первой технологической инновацией для процессоров семейства x86, позволяющей значительно повысить производительность работы с числами с плавающей точкой, 3D графикой и мультимедиа-задачами для современных персональных компьютеров, оснащенных Microsoft Windows. Преимуществами технологии 3DNow! являются высочайшая 3D производительность, более реалистичное и жизненное отображение 3D изображений, полноэкранное Видео и Звук, высокая эффективность работы с Internet. См. описание технологии 3DNow! (engl).
