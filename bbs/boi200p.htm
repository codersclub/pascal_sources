 BOI v2.0 - BBS Onliner Interface   Andrew J. Mead 15.12.1992

Интерфейс для построения BBS Door. Поддерживает коммуникацию непосредственно через COM порт или через Fossil-драйвер. Поддерживает ANSI и AVATAR/1 графику. Может использоваться для много-линейных BBS за счет поддержки DESQview.
BBS Onliner Interface 2.00 (Pascal) Write your own online doors. Source Code included. 18 different drop files, FOSSIL and internal com support. ANSI, AVATAR/1, DESQview, multinode support, too.
