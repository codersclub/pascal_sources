 TriTel Personal Mail Pager 1.1 for BBS  Elm Morrow 29.08.1991

Персональный мэйл-пэйджер для BBS. Сканирует все эхо-арии на предмет наличия в них новых сообщений персонально вам.
The TriTel Personal Mail Pager scans the conference areas and checks for any personal mail. If there is, it will update your personal log on file, or create one if it doesn't exist.

WHAT IT DOES:
        The TriTel Personal Mail Pager scans the conference areas and
        checks for any personal mail.  If there is, it will update your
        personal log on file, or create one if it doesn't exist.

        So far I've checked it with SendNote by Randy Hunt and with the
        Dropped Carried Checker.  It's had no problems.  This IS the
        first version, though, so there's bound to be something wrong.

        If you find a problem with it, please let me know.
