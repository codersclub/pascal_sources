 DirCount 1.1 - File Directory Header Utility  Samuel H. Smith 24.07.1991

Утилита для посчета количества файлов в каждом файловом разделе BBS и их суммарного объема.
DirCount will scan each file directory listing and insert a header that lists the number of files and K bytes used by them. DirCount also generates a summary of all your file listings.

DirCount will scan each file directory listing and insert a header
that lists the number of files and K bytes used by them.

DirCount also generates a summary of all your file listings.
