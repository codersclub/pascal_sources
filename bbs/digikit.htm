 DigKit - Object oriented support unit for Digiboard  Mustang Software 15.07.1992

Объектно-ориентированный модуль для работы с BBS и многоканальными коммуникационными платами типа DigiBoard. Для компиляции требуется пакет Async Professional.
Example Turbo Pascal source for use of Digiboards. Used Async Professional 1.04 from TurboPower Software



12k 
 
CONTENTS
--------

I14BBS.DOC - Specification of INT 14 interface to Digiboard Universal
             driver
APDIGI.PAS - Device layer support unit for Digiboard and Async
             Professional 1.04
OODIGI.PAS - Object oriented layeer support unit for Digiboard and
             Async Professional 1.04

DESCRIPTION
-----------

The Wildcat IM release of Wildcat accesses the Digiboard through the
XIDOS5 Unviersal Driver from Digiboard. The I14BBS.DOC file provided
in this archive contains the description of the interface provided by
the driver. The two file APDIGI.PAS and OODIGI.PAS are example code
of accessing the Digiboard via Async Professional from TurboPower
Software. These files can be used as example code for accessing the
digiboard through the universal driver or for people with Async
Professional these files can be dropped in to add Digiboard support
to that package. 

The structures of Wildcat have not changed for the Digiboard version.
The only difference is the added support of accessing the Digiboard.
