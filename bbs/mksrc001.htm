                              MK BBS Source Code

         This archive contains full modern source code to a BBS.
         Instead of releasing the BBS itself, I have decided to
         release the source code so that sysops can modify it as
         needed to match what they want for their BBS.

         The source code is an unsupported release.  In general I do
         not plan to support or answer questions about MK BBS or the
         source code.

         While I currently am running MK BBS, I am hoping to covert
         soon from running under DOS to running under Linux.

         I am also releasing an archive of executables for the MK BBS
         system.  There are no plans to release the source to the
         utilities (so don't ask).  Many of the utilities use 3rd
         party libraries (like OPRO).
