                        Выпуск # 2
        Предоставляю Вашему вниманию несколько интересных модулей
  для разработки программ на языке Turbo Pascal V 6.0 из библиотеки
  "TURBO PASCAL SUPPORT" (c) 1990 - 1992  Мигач Ярослав.

        Этот выпуск в основном посвящен программистам еще не
  перешедшим к обьектно-ориентированному программированию на
                      Turbo-Pascal.

        Условия распространения в конце этого файла.

  Список файлов на диске :

  REAME.1ST  -  Этот файл

  DEF.PAS    -  Модуль начальных определений транслированный для
                Turbo Pascal V 6.0
                ( Автор приносит свои извинения за допущенные ошибки
                 в исходном тексте программ модуля, опубликованные в
                 прошлом выпуске )

  FKEY11   DOC  - документация на модуль обработки функциональной
                клавиатуры

  FKEY11   PAS  - исходный текст программ модуля обработки
                функциональной клавиатуры

  FKEY11   TPU  - обьектный код модуля обработки функциональной
                клавиатуры

  MENU15   DOC  - документация на модуль организации меню

  MENU15   PAS  - исходный текст программ модуля организации меню

  MENU15   TPU  - обьектный код модуля организации меню

  WINDOW17 DOC  - документация на модуль многооконного интерфейса

  WINDOW17 PAS  - исходный текст программ модуля
                  многооконного интерфейса

  WINDOW17 TPU  - обьектный код модуля многооконного интерфейса

  WINDSHOW PAS  - исходный текст демонстрационной программы

  WINDSHOW EXE  - выполняемый код демонстрационной программы

  WIN      ASM  - модуль подпрограмм низкоуровневого текстового
                  оконного   интерфейса фирмы Borland

  WIN      OBJ  - обьектный код этого модуля.
