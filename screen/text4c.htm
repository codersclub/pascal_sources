 Вывод Текста Растровыми Фонтами для Flat Mode   Ivan Polyacov 18.06.2001

Модуль для вывода текста 4-цветными растровыми шрифтами. Предназначен для работы с любыми буферами в формате 8, 16, 32 bpp при линейной модели памяти (т.е. для TMT, FPC, ...)

Hello Valery,

Еще немного исходников:
на этот раз они предназначены для работы с текстом с помощью 4-цветных шрифтов.
Для BP это делают модули, которые я посылал в прошлый раз,
а сейчас посылаю модуль DirectText, предназначенный для модели Flat.
Кроме того посылаю также редактор шрифтов и несколько примеров самих шрифтов.

Основные преимущества перед использованием стандартных шрифтов
Windows:

1. Можно использовать собственные шрифты, не зависящие от системы.
2. Вывод в любой буфер, нет необходимости в DC.
3. Вывод шрифтов с тенью производится за один проход т.е. быстрее.
4. Возможность градиентной заливки символов.
5. Возможность (если влезть в исходник конечно :) применять любые
   спецэффекты.
