 TxtTool 1.0 - Fonts and Palettes Unit for Text Mode   Thomas Bargholz 24.10.1994

txtTOOL is a Turbo/Borland Pascal 6.0+ library that enables you to use fonts and palettes in text mode. You can display two fonts on screen at the same time, thus giving you access to 512 characters! You can get and set palettes and single colors for the 16 text mode colors! Runs in real and protected mode. Run DEMO.EXE for a quick example!

INTRODUCTION
------------
txtTOOL 1.00 is a (small) library of Turbo/Borland Pascal units (two at the
moment) that enables you to use fonts and palettes in text mode programs on
VGA systems.
txtTOOL 1.00 provides the following features:

- txtFont unit for getting and setting fonts. With this unit you can have up
  to 8 fonts installed in video RAM, and display two fonts simmultanously, thus
  giving you the possibility to display 512 characters at the same time.
- txtColor unit for getting and setting single colors or complete palettes.
- Font Maker visual tool for creating font files for the txtFont unit.
- Palette Maker visual tool for creating palette files for the txtColor unit.
