 Resrs 2.2 - Генератор-редактор экранных форм для TV   Алексей Лагунов 08.04.2001

Генератор-редактор экранных форм в стиле Дельфи для TV. честно говоря он сейчас в связи с переходом с BP на VP не совсем функционален (часть функций урезана коментарями) но основное работает, окошко можно нарисовать.



161k 
 
Это примеры работы с библиотекой UniLib
и примеры проекта, созданного генератором ресурсов Resrs 2.2.
Исходники библиотеки UniLib находятся
<a href="/misc/unilib.zip">здесь. [664k]</a>

Чтобы нормально работать - примеры генератора должны лежать в каталоге
C:\RESRS2 (связано с тем, что в файле проекта жёстко прописывается полный
путь - может в дальнейшем подправлю - если будет нужно).
  расширения файлов:
            *.prj - файл проекта
            *.frm - файл формы (по аналогии с дельфином)

В генераторе принцип работы такой:

Чтобы выбрать объект на форме необходимо щёлкнуть по нему правой
кнопкой мышки - внизу объекта появится курсор - двойная рамка белого цвета
с зеленоватым одинарным уголком. Если теперь взять за курсор и ташить с
нажатой левой кнопкой - то двиаем объект, если тащить за уголов -
то меняем размеры.
Редактор свойств максимально старался сделать похожим на дельфиновский -
единственно - чтобы начать редактирование нужно дважды щёлкнуть по свойству
мышкой, либо нажать пробел (стандартная реакция TListViewer :))

После создания формы - её можно протестировать (нажать F9) - форма
покажется модально.
Чтобы получить исходный текст модуля в меню "Progect" выберите "Build form"
в редакторе создаться исходный текст.

Вот коротко и всё об особенностях ResRs2.
Основные приёмы работы, надеюсь, похожи на дельфин, так что нечему новому
учиться не надо.
