 Broadcast Messages to Background Windows for TV       Unknown 11.04.91


Пример создания двух модальных диалоговых окон в TurboVision
и передачи сообщения для всех фоновых окон.
This program will create two modaless dialog boxes
and will allow only one Dialog2 dialog to be created at any time.
Additionally, it will demonstrate how to broadcast messages
to background windows and update information without changing
the selected dialog.
