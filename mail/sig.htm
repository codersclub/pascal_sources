 Sign Maker v1.0  Денис Нижаде 08.05.1995

Генератор случайных подписей для писем, отправляемых в сеть Релком по протоколу uucp



1k 
 
Organization: Small Venture ECOINFORMSYSTEM

  Это генератор случайных подписей для писем, отправляемых
в сеть Релком по протоколу uucp. В почтовых программах, использующих
uucp (BML, dMail и т.д.) подпись (origin) обычно располагается в файле
personal.sig. Данная программа формирует для каждого сеанса новый
personal.sig, записывая туда случайно выбранную подпись из списка
заранее подготовленных образцов подписей.
