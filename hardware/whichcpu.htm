WHICHCPU 0.43 by Michael Holin
==============================
                                                this is Public Domain


This program is a simple CPU-Identifier for MS-DOS-Machines.
It distinguishes between 386dx and 386sx, 486dx and 486sx and it's
the first program available, which is able to detect Intel's new
RapidCAD CPU.
