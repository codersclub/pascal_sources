Turbo Pascal v2.0


Вторая версия Борландовского Pascal компилятора.

Дата выпуска: 17 апреля 1984


Взгляд на TP2.0 из нашего времени:

      ... в 1984 нам купили CP/M и Turbo Pascal v2.0. Я тогда только окончил школу и начал изучать Pascal. А спустя несколько месяцев у меня появился первый в моей жизни PC, на который я поставил TP, и ... тогда-то все и началось ;-)
      Помню, что TP 2.0 для PC мог генерировать 'очень большие' .COM файлы, больше, чем на 20 k (!) превышавшие обычный лимит. По тем временам это было "огромное" дополнительное количество памяти...
      Поскольку в первых версиях компилятора никакой оптимизации кода еще просто не существовало, мне пришлось придумывать свои собственные методы ручной оптимизации кода для уменьшения размера программ и увеличения быстродействия.
      ... Но TP был *быстрым* ... всего несколько секунд для edit-compile-and-run на моем 8088 компьютере с частотой 4.77 Mhz. В то время мы использовали цикл edit-compile-run, который был очень похож на метод работы с более поздними версиями C++ компиляторов ;-)
Carlos Ribeiro, Tue, 03 Aug 1999

Особенности компилятора:

  • Поддержка процессоров Intel 8088, 8086
  • Поддержка операционных систем CP/M-80, PC-DOS
  • Объем требуемой памяти: 64K.
  • Компиляция только в COM (или CMD) файлы с размером чуть больше 64k
  • Поддержка цветовой палитры CGA
  • Возможность загрузки и запуска внешних .COM программ
  • Управление звуком для PC Speaker
  • Размер исполняемого файла компилятора (вместе с оболочкой и редактором) - всего 36k !
  • Исходники электронной таблицы (Calc), музыкальной "пищалки" (Sound), демонстрация CGA-графики


Состав дистрибутива:


   File             Description
------------------------------------------------------------

TURBO.COM     -  Compiler and Editor
TURBO.MSG     -  Error messages file
TINST.COM     -  Development environment installation program
TINST.MSG     -  Message file for TINST program
TLIST.COM     -  Program lister
CALC.PAS      -  MicroCalc (spread sheet demo program)
CALCMAIN.PAS  -  Root demo program demonstrating include file
                 usage
CALC.HLP      -  On-line manual for MicroCalc
CALCDEMO.MCS  -  Example spread sheet definition
ART.PAS       -  Example CGA Graphics
COLOR.PAS     -  Example CGA Graphics
DOSFCALL.DOC  -  Explanation of MS DOS functions calling
EXTERNAL.DOC  -  Explanation of external programs loading
INTRPTCL.DOC  -  Explanation of interrupt functions calling
READ.ME       -  Read Me File
SOUND.PAS     -  Example Sound program for PC Speaker
WINDOW.PAS    -  Example windowing demo program
