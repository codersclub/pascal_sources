Borland Pascal with Objects v7.0


Borland Pascal v7.0 - самый популярный в конце XX века компилятор Паскаля. Впервые фанаты Паскаля смогли "в одном флаконе" компилировать свои исходники не только для реального режима DOS, но и для 16-битного защищенного режима и даже для Win16 (Windows 3.1)! Впервые Борланд пошла на беспрецедентный шаг - открытие исходного кода всей Run Time Library, за исключением модуля Graph.tpu. Это самый последний Паскаль-компилятор, выпущенный фирмой Борланд. Следующей в линейке компиляторов стала уже Delphi v1.0.

Дата выпуска: 09 марта 1993


Взгляд на BP7 из нашего времени:

  • После выпуска Turbo Pascal 6.0 и Turbo Pascal for Windows 1.0/1.5 "семерка", объединившая и улучшившая эти разрозненные продукты, стала самым продвинутым компилятором Паскаля в 90-х годах XX века. Наконец-то "паскалисты" получили возможность разрабатывать свои программы для защищенного режима и даже для Windows. В дистрибутив входили фактически три продукта: Turbo Pascal 7.0 - дальнейшее развитие компилятора для DOS/Real Mode, Borland Pascal for Windows, и собственно Borland Pascal 7.0 - интегрированный многоплатформенный компилятор.
    Валерий Вотинцев, Исходники.Ru, 2000.


Особенности компилятора:

  • Улучшенная оболочка с подсветкой синтаксиса и неограниченным буфером отката изменений (undo/redo).
  • Оболочка стала многоплатформенной. Для выбора платформы компиляции программы достаточно выбрать соответствующий пункт меню.
  • Компиляция для DOS, DOS Protected Mode (DPMI) и Windows.
  • Бесплатный DOS Extender, для запуска приложений в DOS Protected Mode (DPMI). Возможность преодоления барьера кучи в 640K. Теперь прикладная программа может использовать до 16 MB памяти.
  • Возможность создания DLL (Dynamic Link Libraries) для защищенного режима и для Windows; простота объединения паскаль-программ и библиотек, написанных на C/C++.
  • Для удобства отладки Windows приложений добавлены многофункциональные утилиты Winsight и Winspector.
  • Включенные в дистрибутив исходники Runtime library позволяют досконально изучить принципы работы приложения "изнутри".
  • Поставка на 13 дискетах (5" или 3,5").
  • Стоимость компилятора вместе с исходниками RTL - $195.
