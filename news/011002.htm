  rFido - FTN Data Manipulating Unit for BP7 and VP2+       Dmitry Suhodoev 23.04.2001

Набор различных функций для работы с FTN.
Включает в себя функции:
- IsFidoAddr -is it a fido address?
- ViaDate - date for @Via kludge
- GetBSOName - get bink style outbound file name
- FileBox - get T-Mail filebox name (from t-api)
- HoldFileBox - get T-Mail hold filebox name (from t-api)
- ItAddrMask - Check Address by ftn address mask
- CheckMaskAddress - true if AddrStr falls under Mask
- TruncateAddr - truncate nodes pointzero from addr
- DeTruncateAddr - add nodes pointzero to addr
