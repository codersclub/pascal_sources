 SDS - Sound Deluxe System v5.02c       Maple Leaf 05.11.1997

Sound Deluxe System 5.02с, First public release. Компактная, стабильная и при этом мощная звуковая система
с поддержкой различных звуковых карт и разных звуковых форматов.
SDS (Sound Deluxe System) is a "music interface kit"
designed mainly for ASM+Pascal coders. It is able to load and
play in background a module, using maximum 32 channels which
are mixed software (by mixerHLL) or hardware (by GUS). It is
therefore a good tool in developing and adding stunning music
to your demos and games. It offers you a very easy interface
in Pascal, which you are free (and therefore encouraged) to
use in your programs.
The SDS kernel runs entirely in one 64kb segment (code and own
data). The loaders and the interface code run in another 64kb segment.
