 rTrack 1.1.1 - Ftn Netmail Tracker for OS/2       Dmitry Suhodoev 11.03.2001

rTrack, версия для OS/2 & VP2+.
rTrack - ftn netmail трекер с очень гибкой настройкой.
по возможностям близок к своим эталонам как itrax и itrack:
умеет работать с базами squish, jam, msg (fido/opus) и pkt;
поддерживает outbound как bso, long bso, file box,
t-mail long file box, long file box, brake file box;
поддержка файла роутинга, возможность проверки адресов в нодлисте,
работа с подхабником; возможность настройки красивых шаблонов,
поддержка регулярных выражений при обработке писем,
проверка писем на петлю роутинга;
круглосуточная поддержка автором :-)
