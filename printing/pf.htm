          PF v1.0. (Print File v1.0) Shareware
          для принтеров EPSON, AMSTRAD, СМ

      pf.pas -исходный текст
      pf.doc -документация
      printstr.pas -исходный текст модуля печати строки
      printstr.doc -документация

      СОММЕNTS:

      Unit PrintString;
      interface
        var
          PrinterType:(EPSON800,CM6337,EPSONFX,AMSTRAD); тип принтера
          Quality:(draft,high);                          качество шрифта
          Str:string;                                    строка для печати
        procedure
          ExecPrint;                                     процедура печати


      Запуск:

      pf.exe <имя файла> [/<принтер>] [/<качество>]
      <принтер> - вид принтера: EPSON800, EPSONFX (по умолчанию),
                                AMSTRAD, CM6337;
      <качество> - качество шрифта: 0 - черновое (по умолчанию),
                                    1 - высокое.


      Стоимость эксплуатации: 100 руб.
