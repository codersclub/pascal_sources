   ALGEBRA - Простая реализация матричных операций

   Это Юнит для матричной алгебры. - Годится для любой версии Turbo
Borland Pascal, начиная с 4.0 и даже для GNU Pascal с Turbo Extension.
Представляет собой совершенно прямолинейную реализацию основных численных
методов, но может пригодится тем, кому лень писать самому.
