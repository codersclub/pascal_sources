 Point in Poly - алгоритм "входит ли точка в обьект"   Ivan Kalininskiy 04.10.1999

Описание и исходники алгоритма "находится ли точка в сложном объекте, состоящем из прямых линий"...

   Code by VanDamM // [WRC]

   Алгоритм: Точка лежит внутри многоугольника,
             если кол-во точек пересечения нечетно.
             Все отрезки кроме горизонтальных проверяются на пересечение
             с горизонтальным лучом, выходящим из проверяемой точки.
             При попадании луча в вершину пересечение засчитывается
             только с теми отрезками, выходящими из вершины,
             для которых она является верхней.
