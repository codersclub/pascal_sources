 QCSBPLIB - Math and Graphics Library for BP   David W.Nixon 22.03.1995

Пакет базовых математических программ для BP. Содержит 4 модуля: работа с целыми и вещественными числами; модуль базовых математических функций; модуль расширенной тригонометрии; а также модуль для графического отображения.
Queen City Software Borland Pascal Library contains 4 Borland Pascal UNITs, each with specific functions and duties: SYStem-standard REAL & integer types Unit; basic MATHematical function PACKage; advanced TRIGonometrical function PACKage; basic GRAFical math function PACKage.

This library of Borland Pascal 7.0 source code is hereby dedicated
to the public domain, courtesy of Queen City Software.

N.B.   ALL  UNITs  in  this library  MUST be  compiled with  80x87
instruction generation turned ON.  You may  also want  or need  to
turn 80x87 emulation ON (esp. if you don't have a math coprocessor
installed).  If this condition is not  met, the  source code  will
not compile!

QCSBPLIB  contains  4  Borland  Pascal UNITs,  each with  specific
functions and duties:

     SYSREALU: SYStem-standard REAL & integer types Unit.
     MATHPACK: basic MATHematical function PACKage.
     TRIGPACK: advanced TRIGonometrical function PACKage.
     GRAFPACK: basic GRAFical math function PACKage.
