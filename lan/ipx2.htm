 IPX Unit and a Simple Chat Routine   Alex Frounze 27.09.1999

Обмен информацией между двумя или большим количеством компьютеров по локальной сети с использованием протокола IPX. Модуль + чат-программа.



6k 
 
                             -= In Russian =-

  Здесь предсатвлен простой модуль для обмена информацией между двумя и  более
(до 10-ти) компьютерами по локальной сети. Используется стандартный IPX прото-
кол. Если Вы работаете в Windows, проверьте в  настройках  сетевого  окружения
присутствие протокола IPX. При отсутствии, добавьте и перезагрузите компьютер.
  Как пример прилагается примитивная программа chat.

                                                 Автор: Алексей Фрунзе

                             -= In English =-

  Here is shown simple unit for information interchange between 2 and more (up
to 10) computers via local network. Unit uses standard IPX  protocol.  If  you
work under Windows, check if there is installed IPX protocol  in  the  Network
Neighbourhood properties. If not, add it and reboot your computer.
  Very simple chat program is an example of using the unit.
