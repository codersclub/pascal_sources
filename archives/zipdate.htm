Программа для установки даты создания ZIP файла в соответствии с датой создания упакованных внутри файлов.

      Hi!

      Предлагаю вашему вниманию программу, которая изменяет дату

пригодилась. Но у нее был один недостаток: При перепаковке архивов
создаются директории с текущей датой, и именно эта дата и подставлялась
в дату .ZIP файла :-(. Поэтому пришлось подкорректировать программу:
теперь она, извините, плюет на даты создания директориев. В учет
принимаются только даты ФАЙЛОВ.

This is a quick little utility that will scan all your ZIP files and set
the date stamp of the file to the latest date of any file contained
within the ZIP.  (confused?)

Calling Syntax is:  ZipDate filespec

  eg.  ZipDate *.Zip

Credits:  Most of the Zip file functions were adapted from the Zipper
Unit written by Tom Guinther - Thanks Tom!
