What is it:
===========

CompSys is an toolbox for examining compressed files. Using it,
you can look into compressed files for the contained filenames,
check for security envelopes (if available) check out
selfextractors and even a Mac compressed fileformat!

I wrote two of these systems before, but was never really
satisfied. This version is the first one for which I feel
confident enough to release it as shareware.

But before we go into details, here's an overview of the
compressors that CompSys can handle:


  Compressor  |Recognised |SelfExtr.|Sec. Env.| FileExtra field
