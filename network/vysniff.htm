 VyChat Sniffer for Delphi   Coban 18.10.2001

Сниффер (перехватчик пакетов) для известного Vypress Chat'а, отслеживает все личные сообщения в сети и строит удобные для чтения логи. Это своеобразный пример работы с драйвером PCap. Компилируется в Delphi, может быть переделан для FPC, TMT или VP.



735k 
 
  Программа "вынюхивает" личные сообщения рассылаемые по сети и
сохраняет их на диске. Т.е. Вы сможете увидеть всю переписку локальной
сети :)

  VySniff'у можно передать некоторые параметры (этакое управление с
  коммандной строки)

  vysniff.exe start или просто vysniff.exe -- запуск
  vysniff.exe close -- закрыть
  vysniff.exe status -- посмотреть запущена ли программа

  При работе приложение создает каталог для каждого пользователя,
который участвует в переписке, например каталог `Вася', внутри
создаются файлы с именами тех людей кому пишет `Вася' или от кого он
принимает сообщения (Коля, Петя...),
если файл внутри каталога с префиксом _pvt -- то тут содержится личная
переписка по чату (личный чат, не сообщения).


  !!Для корректной работы необходим драйвер PCap!
  Он прилагается в архиве.
  Самую свежую версию его можно укатать с
  http://netgroup-serv.polito.it/winpcap

  В коммандной строке программе можно указать сл. параметры:
    start -- стартовать
    close -- закрыть
    status -- вывести статус программы (работает или нет)
