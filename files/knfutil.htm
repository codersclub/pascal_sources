 Маленькие файловые утилиты  Никита Корзун 05.05.1992

Файловые утилиты: Создание файла с именем системной даты; Создание и уничтожение каталога; Определение свободного и занятого места; Определение секунд в дате создания файла; Поиск программы по всему диску и запуск ее; Вытаскивание параметров клавиатуры.

                       Вариации на разные темы

                                KNCF
В текущей DIR создает файл-пустышку с именем системной даты. Например:
6 apr.92. (Идея и реализация в MultiEdit Корзуна Е.Г.).

                                KNDEL
Очень опасная и ядовитая  вещь. При запуске без  параметров уничтожает
все в текущей директории  со всеми поддиректориями, ничего  не говоря.
Можно  запускать  с  параметрами  -  именами  SUB-DIR, тогда уничтожит
только их. Работает и в корне, кроме диска C:, где ее и надлежит  ради
безопасности держать. Для использования необходимо переименовать  файл
KNDEL.EX в  KNDEL.EXE (Идея  и первоначальная  реализация Сидельникова
С.С. - DELC.EXE)

                                KNDI
Наглядная информация о свободном и занятом месте на всех дисках.

                                KNDIR
Иногда бывает полезно посмотреть  на секунды во времени  создания фай-
лов. Возможный параметр - маска файлов.

                                KNDD
В  текущей  DIR  создает  SUB-DIR  с  именем системной даты. Например:
92_05_05. Бывает полезно.

                                KNFR
Ищет  файл  по  всем  дискам  и  директориям,  и, если находит, идет в
директорию,  где  нашла,  там  запускает,  говорит  Exit Code и уходит
откуда вызвали. Поиск  сначала идет по  PATH, а потом  по всем дискам,
начиная с C:. В качестве параметров дается имя выполняемого (EXE,  COM
или  BAT)  файла,  можно  без  расширения  (в  этом  случае   работает
стандартный досовский приоритет - сначала  COM, потом EXE, а уж  потом
BAT), и его параметры. Не надо помнить где сидит TETRIS!

                               KNKBDI
Вытаскивает из клавиатуры все, что только можно. Очень наглядно.
