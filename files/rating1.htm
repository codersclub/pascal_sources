 4 Файловых Утилиты  С.Свердлов 18.05.1992

4 программы: Просмотр дерева каталогов, Удаление подкаталога вместе с файлами, Удаление .BAK файлов с диска, Рейтинг популярности запуска программ



50k 
 
            Вы  становитесь  участником  второго  этапа  эксперимента  по
       определенияю рейтинга  популярности программ   для IBM-cовмеcтимых
       компьютеров с  помощью программы RATING.

                 В комплект программы RATING входит :

       README            Этот файл;
       START    BAT      Запуск программы RATING;
       RATING   EXE      Программа определения рейтинга;
       RATING   DOC      Описание программы RATING, адрес, реклама;
       RATING   HLP      Инструкция к программе RATING;
       RATING   LST      Файл рейтинга;
       MERGE    EXE      Программа слияния файлов рейтинга;
       ST       EXE      Просмотр дерева каталогов;
       ST       PAS
       CT       EXE      Подсчет числа байт в файлах дерева каталогов;
       CT       PAS
       KT       EXE      Уничтожение текущего каталога с подкаталогами;
       KT       PAS
       BT       EXE      Удаление BAK-файлов с текущего диска;
       BT       PAS

          Программы КТ  и  ВТ,  удаляющие  файлы,  требуют  подтверждения
       выполнения потенциально  опасных действий.  Их можно запускать, не
       опасаясь несанкционированного уничтожения данных.
          Для вывода данных о дереве каталогов в файл с целью дальнейшего
       их просмотра  программы ST и CT можно запускать с перенаправлением
       выходного потока:
                                     ST > файл
                                     CT > файл

          Исходный текст программы RATING будет опубликован по окончании
       эксперимента.
