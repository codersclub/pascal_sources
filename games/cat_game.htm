Игры в Графическом Режиме

 Cat - Birthday Game       Pavel Anokhin 15.02.1999


Графическая игра-поздравление "Кот", режим 13h, Real Mode.
Черный кот бегает влево-вправо и ловит пробегающих мимо (сверху-вниз) мышей.
Напоминает идею старинной игрушки "Война с марсианами".
В качестве награды игрок получает поздравление с днем рождения.
