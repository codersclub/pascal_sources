Игры в Графическом Режиме

 GMS - 2D Graphical Game 320x200x256       Dark-Cloud Team 30.08.1997


2-х мерная графическая игра. Танкетка ездит по полю,
на котором расположены препятствия, мины, сюрпризы и вражеские орудия. Управление движением самой танкетки и орудийной башни - раздельное. Танкетка управляется с помощью клавиш,
а поворот орудийной башни и стрельба - с помощью мыши.
A great game for mode $13 by dark-cloud,
powered by GMS - Great Map Scroller Engine by dark-cloud 1997,
and GamePro v2.41 Unit by Ariel Keselman 1995,96.
Additionally included: Level Editor and Sprite Editor (16x16, 32x32).
