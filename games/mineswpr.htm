 Minesweeper for DOS  Mudbutt 04.09.1993

Minesweeper for DOS! The DOS version of the popular Windows game of mind-bending logic. EGA/VGA, Pascal source code included.



9k 
 
The enclosed files are a DOS version of the popular game Minesweeper and the
pascal source code for it, written by a friend of mine.  When I asked if he
minded me uploading this to the list, he told me that it would be okay.  He also
said that since most of this was developed at work (we both work for the
government) it was probably, by definition, public domain. (All software
developed by the government is public domain.)  ;->

So, this stuff is released to the public.  If you make any changes/improvements,
please be sure and post them to the list.

Ciao,
Mudbutt
