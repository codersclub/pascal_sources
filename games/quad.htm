 Quadruple Memory 1.3 - Memory Trainer   Justin Pierce 19.12.1997

Игра для тренировки памяти. На столе колода перевернутых карт. Задача - с помощью мышки найти и запомнить местоположение четырех одинаковых карт. BGI-графика.
Memory training card game. Guess the location of four cards.
