 FIL. Как открыть на запись ReadOnly файл   Sly Golovanov 27.12.1997

Изменение атрибутов ReadOnly файла для дальнейшей записи в него.


 
 
              Pascal FAQ created by SLY Golovanov, 2:5020/278.13
              (slightly corrected by Valery Votintsev 2:5021/22)
==============================================================================

Q: 1) А как открыть на запись файл с атрибутом read/only? Ругается.
   2) Как удалить r/o файл?

A: Hужно снять атрибут r/o с файла, а затем делать с ним все, что
   заблагорассудится.

Uses DOS;
var
  t:text;
  attr:Word;
Begin
  Assign(t,'file');
  GetFAttr(t,attr);          {Прочитать атрибуты файла }
  attr:=attr and $FE;
  SetFAttr(t,attr);          {Сбросить атрибут ReadOnly}
  Append(t);
  Writeln(t,'Hello!');       {Дописать что-либо в файл }
  Close(t);
  attr:=attr or $01;
  SetFAttr(t,attr);          {Восстановить атрибут ReadOnly}
End.
