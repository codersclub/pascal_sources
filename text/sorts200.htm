 SortUtil 2.0 - Маленькая Утилита Сортировки   Alexander Boychenko 02.02.1996

SortUtil v2.0. - Маленькая утилита сортировки, написанная для использования в связке с PostUtil. Поддержка русского языка, "оконной сортировки" и т.д.



5k 
 
SortUtil v2.0 by Alexander Boychenko.

Usage:  SortUtil filename.ext [-switch1 [-switch2 [...]]]

Switches:
-A    - Не надо ни русского, ни латинского UpCase
-B    - Не сохранять исходный файл
-Cxxx - Рассматривать, начиная с xxx колонки
-E    - Запретить выдачу сообщений об ошибках
-Fxxx - Длина номеров строк - ххх
-Lxxx - Длина рассматриваемой строки - xxx
-M    - Запретить выдачу всех сообщений
-N    - Надо нумеровать строки
-R    - Не надо русский UpCase
-Sxxx - Пропустить первые xxx строк в файле
-T    - Не надо латинского UpCase
-V    - Обратная сортировка
-Wxxx - Обрабатывать только xxx строк (от первой не пропускаемой)
-Xwww - Максимальное кол-во строк, которые нужно считать из файла
        (остальные "обрежутся")
-Ywww - Номер первой, считываемой из файла строки (остальные "обрежутся")
