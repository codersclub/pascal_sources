 EdVert - Вертикальный Редактор  Павел Северов 12.09.1991

Программа предназначена для вертикального редактирования текстовых файлов (действие команд относится к одним и тем же позициям всех строк)

Форма обращения:

    EDVERT  PROGRAM.EXT  OTKUDA.EXT  KUDA.EXT

       PROGRAM.EXT - файл содержащий команды для EDVERT;
       OTKUDA.EXT  - входной файл;
       KUDA.EXT    - выходной файл.

   Программа предназначена для вертикального редактирования текстовых
файлов (действие команд относится к одним и тем же позициям всех строк)

  Команда должна начинаться с первой позиции строки, конец строки служит
разделителем между командами. Пробел в первой позиции является признаком
строки - комментария

                         Команды EDVERT:

= <нач.поз.> <кон.поз.> <комментарий> - определение столбца текста,
                   без изменений попадающего в выходной файл;

+ <некий текст>  - данный текст будет вставлен во все выходные строки
                   в соответствующей позиции;
