                              Часть 1.

---------------------------
BackSp   - Обрезание "хвостовых" пробелов в текстовом файле;
           Некоторые текстовые редакторы  ( в частности, ФОТОН ) при обра-
           ботке прямоугольных блоков текста  добавляют к коротким строкам
           соответствующее количество пробелов.  Возможно, ничего страшно-
           го в этом нет, но меня раздражает, когда я после этого, нажимая
           клавишу End, вдруг "улетаю" куда-то далеко за край экрана. Кро-
           ме того, мне  просто жалко сотни байт, съедаемой  этим мусором.
           Программа BackSp "обрезает"  хвостовые пробелы в каждой строке.

---------------------------
DelPage  - Уничтожение постраничной разбивки файла;
           Программа, функционально противоположная VesPager  (см. ниже).
           Позволяет уничтожить в текстовом файле предварительную разбив-
           ку на страницы.

---------------------------
PosFile  - Позиционирование строк текстового файла;
           Производит  сдвиг каждой строки текстового файла вправо (влево)
           на указанное число позиций  начиная с заданной  позиции.  Может
           использоваться при разбивке файла для печати в две колонки.

---------------------------
CodUkrMS - Перекодировка украинских символов из стандарта
           MicroSoft в украинский и обратно;
