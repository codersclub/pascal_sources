 Texture Mapping Algorithm Example [320x200x256]   Alex Frounze 09.05.1999

Демонстрационная программа - текстурированый вращающийся куб. Демонстрирует возможные варианты оптимизации по скорости. Описание:
- Выбор метода текстурирования
- Вкл/выкл ожидания обратного вертикального хода луча
- Возможность приближения и удаления куба
- Метод разбиения скан-лайна на на группы по 8/16 пикселов и линейной интерполяцией (оптимизировано для Pentium).
The Perfect Texture Mapping Algorithm Example. Final release. Numeric processor is requred. All bugs have been fixed
