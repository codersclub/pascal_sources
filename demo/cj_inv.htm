 для компиляции интрошки:

  в директории \MAKE_PAK лежат файлы для создания pak файла
  надо запустить make_pak.bat
  появится файл wrc_blur.pak - его нужно скопировать в директорию интрошки

  для компиляции интрошки нужна команда:

  c:\bp\bin\bpc.exe -B -$G+ -$N+ wrc_blur.pas
  ^^^^^^^^^^здесь ваш путь к компилятору
