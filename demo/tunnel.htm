Texture mapped tunnel effect
          by Gedge

                 

Well, a few ppl have asked me for the code to this & I have always
refused but realised the other day that there was no point in keeping
it to myself as I learned how to do it from an article in a mag by
Sandman/Valhalla so I was being a little hypocritical :)

So here it it.  It's far from perfect as there are 2 obvious bugs in it.
The 1st is a sort of cross effect running through the tunnel (which I
have seen in a few other tunnels so it's a common bug :) and the other
is that the perspective is fucked.  It looks like it's kinda "squirting"
in the distance :(

If anyone figures out how to fix these problems, please send me the
solution :)

This was my last ever program in Pascal & I have now moved onto Watcom C
so maybe I'll release some lame C code in the future too ;)

Anyway, anyone that might wanna send me a solution to the bugs or just
generally talk rubbish then you can mail me at gedge@eh.u-net.com :)

FOO & all the other assholes on Euro & US #coders that I know ;)
