Модуль INIFILES содержит три функции:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
function GetProfileString( section:string;  entry:string;
                      default:string;  filename:PathStr):string;

function GetProfileInt( section:string;  entry:string;
                     default:integer;  filename:PathStr):integer;

function WriteProfileString( section:string;  entry:string;
                      value:string;  filename:PathStr):boolean;

***************************************************************
ОПИСАНИЕ

+-------------------------------------------------------------+
|    function GetProfileString(section:string; entry:string;  |
|                  default:string; filename:PathStr):string;  |
+-------------------------------------------------------------+
Функция GetProfileString возвращает текстовую строку, ассоциированную
с индексной строкой entry в секции section файла filename.

Параметр               Описание
----------------------------------------------------------------
section                 содержит имя секции
entry                   содержит имя индексной строки
default                 содержит строку, которая будет возвращена,
                        если по каким-либо причинам не найдена
                        индексная строка.
filename                содержит имя инициализационного файла

Возврат
~~~~~~~
Функция возвращает текстовую строку, ассоциированную с индексной
строкой entry
