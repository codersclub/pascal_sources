 L-LIB - Библиотека Текстовых Интерфейсов под ДОС   Victor Ostashev 04.02.1996

Паскаль: Библиотека для создания интерфейсов программ под ДОС (текстовый режим): окна, меню, сохранение экрана, клавиатура. Полные исходные тексты.



32k 
 
                       Библиотека дла Borland Pascal

      Настоящая библиотека распространяется на основании LGPL и может быть
использована как для написания некоммерческих, так и для написания коммер-
ческих продуктов. Никаких  обязательств  автор  на себя не берет и никакой
ответственности за  ущерб, связанный с использованием или неиспользованием
настоящей библиотеки не несет.

      В состав библиотеки входят следующие модули (в исходных текстах):

      Calendar.pas - набор функций для работы с датами и вычислений по ка-
                     лендарю.
      Dosfile.pas - набор функций для открытия файлов на низком уровне.
      Dospath.pas - обработка путей.
      Inifile.pas - работа с текстовыми конфигами.
      L_clock.pas - при подключении этого модуля в правом верхнем углу эк-
                    рана будут висеть часики.
      L_edit.pas -  простейший текстовый редактор, встраиваемый в программы.
      L_emnu.pas - один из двух модулей, создающих меню.
      L_kbd.pas - работа с клавиатурой.
      L_misc.pas - всякая всячина.
      L_mnu.pas - еще один модуль меню.
      L_mouse.pas - мышка, естественно.
      L_rwscr.pas - запись экрана в файл и чтение экрана из файла.
      L_scr.pas - экран, понятное дело.
      Matchwld.pas - сравнение с шаблоном.
      Matchwld.obj - это obj к нему.
      Tempfile.pas - создание временного файла.

      Подробное описание функций содержится в комментариях в исходниках.
