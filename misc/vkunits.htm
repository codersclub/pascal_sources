Отправляю обещанный файл.
Тут поправленые юнитки - без них ни одна программа моя работать не будет :)
Щас распишу что какой файл значит, возможно, некоторые я вообще не правил,
но большинство всё-же покорежил. Будут вопросы - мыль. Напишу потом
маленькие экзамплы для использования данных юнит.

SCREEN   ZIP  Сжат IMPLODE  OBJ, как это делать - см. zip.pas
FORMAT   ASM  Просто вытащил функцию FormatStr - для Windoze
IMPLODE  OBJ  Объектник зипирования/расзипирования -  см. zip.pas
APP      PAS  Поправленый, подробности внутри
ARC      PAS  Куча полезных функций и процедур
BROWSE   PAS  Типа TListBox'a - еще на 5.5 написан, не использую уже
CALC     PAS  Калькулятор правленый
COLORSEL PAS  Вот это не правил
COLORTXT PAS  А-а классная фича - TStaticText любым цветом
DBSERV   PAS  Сервис для баз данных - только читать
              Пришлю потом свой DBVIEW на TV
EDITORS  PAS  Печатать его научил
ENDECODE PAS  тут ясно все
FINDFILE PAS  поиск файлов по подкаталогам - из енглицкого СВАГА, но
              там баг был внутри - под Осью криво работал - я пофиксил
FORMSTR  PAS  Ага FormatStr
GADGETS  PAS  Вроде не правил
HELP     PAS  Тут константы всякие, черт знает может где в юнитках
              они и упоминаются
HELPFIL$ PAS  Оригинальный
HELPFILE PAS  Сделано Alt+F1 и кое-какие баги Ворландовские пофиксены
HIDEDIR  PAS  Ставит Хидден на директории
INIFILE  PAS  Вот два файла для работы с ини
INISERV  PAS
INPLONG  PAS  PInputLine Для чисел вроде - уже не помню
MSGBOX   PAS  Подправил - стрелки вправо/влево и правая кнопка мыши = Enter
OUTLINE  PAS  Стандартный
PROTECT  PAS  Оой, еще сохранилось - это я хотел защиту от копирования
              сделать
REBOOT   PAS  Ясно что
SCROLLB  PAS  А-а, клевая фича - PListBox, который вправо-влево гоняется
STDCONST PAS  Стандартные константы мои
STDDLG   PAS  Пофиксен
STRINGS  PAS  Стандартный
SUMIL    PAS  Там какой-то объект для ввода суммы прописью
TIMEOUT  PAS  Во - тут наследник от ТApplication
TOOLS    PAS  Куча полезных функций, процедур и объектов
VALIDATE PAS  Стандартный
WINISERV PAS  Тоже ини-сервис для Виней - мне их не понравился :(
WINMAIN  PAS  А, что-то
